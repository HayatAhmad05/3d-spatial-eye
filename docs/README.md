# Documentation

This directory contains additional documentation and assets for the 3D Spatial Eye project.

## Images

The `images/` directory is for:
- Screenshots of the web interface
- Wiring diagrams
- Demo GIFs
- Hardware photos

## Adding Screenshots

To add screenshots to the main README:

1. Place your image in `docs/images/`
2. Reference it in the README using:
   ```markdown
   ![Description](docs/images/your-image.png)
   ```

## Suggested Images

- `demo.gif` - Animated demo of the scanner in action
- `web-interface.png` - Screenshot of the 3D viewer
- `wiring-diagram.png` - Hardware connection diagram
- `hardware-setup.jpg` - Photo of the assembled scanner
